ABP (pronounced A-B-P) is an open-source, opinionated application framework for building modular, domain-driven web applications with .NET. It provides ready-made architectural layers (modules, domain, application, presentation), built-in features like identity, localization, multi-tenant support, background jobs, and UI tooling, so developers can focus on business logic rather than boilerplate. ABP emphasizes clean architecture, modularity, and extensibility, making it ideal for enterprise‐grade projects that need to scale, evolve, or be composed from interchangeable parts.
